# Folio — 브라우저 전용 PDF 병합 · 편집기

여러 PDF를 불러와 **페이지 삭제 · 회전 · 순서 변경**한 뒤 하나의 PDF로 병합해 저장하는 단일 파일 웹앱입니다. 설치가 필요 없고, 모든 처리는 **브라우저 안에서만** 이루어집니다 — 불러온 파일은 어디에도 업로드되지 않습니다.

## 사용법

[`index.html`](index.html)을 브라우저로 열기만 하면 됩니다. (또는 GitHub Pages로 배포)

- **불러오기** — 드래그앤드롭 또는 파일 선택으로 여러 PDF를 한 번에
- **편집** — 페이지별 90° 회전 · 삭제, 체크박스로 다중 선택 후 일괄 처리
- **순서 변경** — 페이지 카드를 끌어서 자유롭게 재배치 (파일 경계 무관)
- **저장** — 현재 순서와 회전 그대로 하나의 PDF로 내보내기

### 단축키
- `Ctrl/Cmd + A` — 전체 선택
- `Delete` / `Backspace` — 선택 페이지 삭제
- `Esc` — 선택 해제

## 기술

- [pdf.js](https://mozilla.github.io/pdf.js/) — 페이지 미리보기 렌더링
- [pdf-lib](https://pdflib.js.org/) — PDF 병합 · 회전

두 라이브러리는 CDN에서 불러오므로 **첫 실행 시 인터넷 연결이 필요**합니다. 완전 오프라인 환경에서는 두 `.js` 파일을 내려받아 `index.html`의 `<script src>`를 로컬 경로로 바꾸면 됩니다.
